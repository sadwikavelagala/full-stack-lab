Understanding Hyperlinks in HTML

The <a> tag is used to create hyperlinks in HTML. The href attribute specifies the URL of the page the link goes to.

Basic Hyperlink

Visit Example.com

This link will open the website example.com in the same browser tab.

Opening Link in a New Tab

Go to Wikipedia

The target="_blank" attribute makes the link open in a new browser tab or window.

Other Target Options

  • target="_self" (default) - Opens link in the same tab.
  • target="_blank" - Opens link in a new tab or window.
  • target="_parent" - Opens link in the parent frame.
  • target="_top" - Opens link in the full body of the window.

Relative Link

Go to Page 2 (relative link)

This type of link points to another page in the same website folder.
